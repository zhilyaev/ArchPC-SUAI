# Краткий F.A.Q. по командам VAX-11

## ACB: Изменение, сравнение, переход
```C
ACB (x,y,z,j){
    z = y+z
    if (y>=0 & z<=x) | (y<0 & z>=x)
        JMP J
}
```
### Пример использования

| 4 | 00  |  HALT | |

* Что происходит в ШАГЕ 1? 
**СРАВНЕНИЕ**
```C
ACB(0, 0, A[i]){
    z = 0 + A[i]
    if(0>=0 & z<=0)
        goto ШАГ 3
    else goto ШАГ 2
}
```
* Что происходит в ШАГЕ 2? 
**ИЗМЕНЕНИЕ**
```C
ACB(SUM, A[i], SUM){
    SUM = A[i] + SUM
    // Условие выполняется ВСЕГДА
    if(A[i]>=0 & SUM<=SUM) | (A[i]<0 & SUM>=SUM)
        goto ШАГ 3
    else goto ШАГ 3
}
```
* Что происходит в ШАГЕ 3? 
**ПЕРЕХОД**
```C
ACB(0, -1, i){
    i = -1 + i
    if (-1<0) & (i>=0)
        goto ШАГ 1
    else HALT;
}
```
